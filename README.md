# Codex Usage Checker

一个最小可用的 Chrome / Edge 扩展，用来从当前打开的 OpenAI 页面里抓取可见的 Codex 用量信息。

## 当前实现

- 支持 `chatgpt.com` 和 `openai.com` 页面
- 在弹窗里显示：
  - 剩余用量
  - 已用 / 总量
  - 重置时间
- 额外展示命中的文本线索，方便你判断页面抓取是否准确

## 安装

1. 打开浏览器扩展管理页
2. 开启“开发者模式”
3. 选择“加载已解压的扩展程序”
4. 选择当前目录：`C:\Users\neoso\VSCodeProjects\codex_usage`

## 使用

1. 先打开一个能看到 Codex 用量信息的页面
2. 点击扩展图标
3. 如果结果不对，点“重新扫描”

## 说明

这个版本没有依赖私有 API，而是直接从页面可见文本中提取额度信息，所以：

- 好处是不用额外登录授权
- 风险是页面文案或 DOM 结构变化后，匹配规则可能要更新
